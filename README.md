1.  **Clone the repository**
    ```bash
    git clone https://github.com/Tsoorh/tiran-frontend-fixed
    cd tiran-frontend-fixed
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

4.  **Run the application**
    ```bash

    ```
